## Executive Summary

本文提出了 BOUNDARY_SYNC 通信协议与 CAF 耦合度量，旨在测量多智能体 LLM 中通信引起的表示同质化。其在 GPT‑4o 上进行了受控消融、跨模型复制及动态分析，发现文本通信导致同质化，并声称图像通信呈现相反的多云化模式。然而，图像实验严重依赖错误的基线——所有图像条件的 CAF 均以文本孤立基线为分母，致使图像 CAF 反映的是任务间固有差异而非通信效应，整条“模态不对称”核心主张失去证据支撑。同时，论文未提供任何可访问的代码、数据或 prompt 细节，关键数学公式以缺失图片呈现，生成参数完全缺失，可复现性为零。基线对比仅与 DeGroot 模型比较，未纳入现有 SOTA 多样性度量，且跨模型协议不统一。整体而言，论文在实验设计、可复现性与基线对比上存在不可修复的根本性缺陷，即使有一定消融设计和复制尝试的亮点，也不足以挽救其致命错误。

## Detailed Assessment

### Major Strengths
- 设计了一个干净的通信消融（no-sync）作为关键控制，成功剥离了通信对文本同质化的驱动作用，为因果推断提供了直接内部证据。
- 提出了标准化 CAF 度量并配套 Bootstrap CI 与 BH-FDR 校正，为多智能体 LLM 耦合度量提供了可操作的框架。
- 尝试跨模型（GPT-4o、DeepSeek V4 Pro、Qwen3.7-Plus）复制，并在 Qwen 上引入了无关文本扰动控制，部分排除了提示长度混淆。
- 动态 toggle 实验揭示耦合的无状态特性，与经典 DeGroot 模型的单调收敛形成对比，对后续系统设计有启发意义。

### Major Concerns
- 【致命】图像实验完全依赖错误的基线：所有图像条件（包括 no-sync）的 CAF 分母均为文本孤立基线 C1，而非图像自身的独立基线。这导致图像 CAF 的值反映的是任务间固有分布差异，而非通信引起的耦合或分散。论文据此宣称的“图像通信呈现相反的多云化模式”及“模态不对称”核心主张均无有效证据支持，整个图像部分的结论不可信，直接动摇了论文的完整论证。
- 【致命】完全不可复现：论文声称所有代码、数据、分析脚本存于 project repository，但未提供任何链接或标识符，审稿人无法访问。所有生成参数（temperature、top_p、max_tokens、seed）、完整 prompt 模板、系统消息等完全缺失，加之关键数学模型以图片形式省略，导致即使获得代码亦无法重建任何一次 API 调用。这直接违反可复现性的基本要求。
- 基线对比严重缺失：未与已有 LLM 输出多样性/耦合度量（余弦相似度、有效秩、self‑BLEU 等）进行直接比较，无法证明 CAF 的增量价值；仿真基线仅限于 DeGroot 模型，缺少 Hegselmann‑Krause 等其他经典意见动力学模型；跨模型复制面板过窄且测量协议不统一，难以支撑结论的普适性。

### Minor Issues
- GPT‑4o 文本同步条件仅有 N=30 单次运行，样本量不足且未进行无关文本扰动控制，排除提示混淆的证据不完整。
- 跨模型测量协议不统一（GPT‑4o 使用句子嵌入分箱 JSD，DeepSeek/Qwen 使用直接类别概率分布），导致绝对 CAF 不可直接比较，削弱跨模型效应大小的定量论断。
- CAF 比率估计的有限样本偏差虽被承认但未作任何校正（如 Jackknife 或偏差修正），在小样本下置信区间覆盖可能不足。
- 图 4 索引与实际标记存在歧义，关键公式以不可见图片展示，损害了阅读和复现的清晰度。

## Revision Suggestions

1. 修正图像实验基线：为图像任务单独设置无通信独立基线（图像 NoSync），以其 JSD 均值作为分母重新计算所有图像条件的 CAF，并重新评估“模态不对称”及统计显著性。不完成此项修正，图像部分完全无效。
2. 提供完整的可复现包：在 camera‑ready 版本中给出可公开访问的代码仓库链接、所有原始数据（JSON/CSV）、完整的 prompt 模板（系统消息、neighbor blending 插入逻辑）、以及所有 API 调用参数（temperature, top_p, max_tokens, seed 等）。这是任何重新审稿的前提。
3. 增加与现有度量的直接比较：在相同数据上计算余弦相似度、有效秩、self‑BLEU 等代表性指标，并与 CAF 进行相关性和排序一致性分析，以证明 CAF 的独特价值或互补性。
4. 统一所有模型的测量协议：要求所有模型在相同任务、相同嵌入空间（如统一用 sentence‑transformer 分箱 JSD）下输出，避免因输出格式差异引入的不可比性；同时控制推理参数（如保持默认思维链设置一致）。
5. 补充 GPT‑4o 文本同步条件的干扰控制：仿照 Qwen 实验，在 GPT‑4o 上增加等长无关文本扰动条件，以排除提示长度和复杂性对同质化的混淆。
6. 修正 CAF 比率估计偏差：对 CAF 使用 Jackknife 或 Delta 方法进行偏差校正，提供偏差修正的置信区间，并分析其对小样本条件下结论的影响。
7. 增加仿真基线：至少再纳入一种经典意见动力学模型（如 bounded confidence 模型），并讨论不同理论参照对现象解释的影响。

## Technical Discussion

本文的根本脆弱点在于基线定义的逻辑不一致性，这直接导致图像实验的完全崩塌。CAF 被设计为度量通信引起的耦合强度，分母必须是同任务、同模态下的独立基线。然而，C5 和 C6 等图像条件直接复用了 C1（文本无关响应）的 JSD 均值作为分母，这相当于假设文本与图像任务的响应分布天然可比，且独立状态下分散度相同——这是一个既无检验、也无理论支持的强假设。更严重的是，C5 NoSync 本身是一个图像的独立条件，完全可以用自身的 JSD 作为分母来定义图像基线，而论文却选择了跨任务比较。因此，C5 的 CAF>1 很可能是图像描述任务本身比文本段落任务具有更高的表达多样性，而非通信导致的多云化。这一错误使得整条“模态不对称”和“基于基线水平的解释”都建立在沙地上，撤去这根支柱，论文的完整性随之倒塌。若论文试图修补，不仅要重构图像基线，还需在下游重新检验所有图像相关的统计结论，包括与 DeGroot 仿真的对比和模态差异的成因——从根本上是一次推倒重来的修正。鉴于这一缺陷的严重性和不可简单修复的特性，论文在现行状态下不具备任何被接收的可能。
